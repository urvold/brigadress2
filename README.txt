
# БригАдрес — статический сайт

Готов к загрузке на GitHub Pages. 
1) Загрузите содержимое папки в ваш репозиторий (`main` / `docs`).
2) Включите **GitHub Pages** (Settings → Pages → Deploy from branch → main / root).
3) Положите свои изображения:
   - `assets/projects/` — превью проектов (замените `barn123.jpg`, `loft61.jpg`, `barn145.jpg`, `right103.jpg`).
   - `assets/partners/` — фото для карусели (1.jpg … 6.jpg).
   - `assets/reviews/` — фото для отзывов (1.jpg … 4.jpg).
4) Отправка формы сейчас настроена через `formsubmit.co` на `brigadres@mail.ru`. 
   Если хотите EmailJS, откройте `index.html`, замените форму на свою интеграцию.

Цвета и стилистика взяты из логотипа. На мобиле есть фиксированная кнопка «Получить 3–4 предложения».
